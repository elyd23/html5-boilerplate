Blueprint + HTML5boilerplate - test pages

Welcome to the Blueprint test pages. I forked the original blueprint files and combined them with the HTML5boilerplate. This is Version is only a draft. Please add your toughts. Thx. The HTML files below tests most HTML elements, and especially classes provided by Blueprint.

Test page Main files tested Description
Sample page grid.css, blueprint-html5boilerplate-style.css A simple sample page, with common elements.
Grid grid.css Tests classes provided by grid.css.
Typography Style moved blueprint-html5boilerplate-style.css Tests HTML elements which gets set in typography.css.
Forms forms.css Tests classes and default look provided by forms.css.

Note about the compressed versions: These test files utilize the compressed files. In other words, if you change any of the source files, you'll have to re-compress them with the ruby script in the scripts folder to see any changes.
